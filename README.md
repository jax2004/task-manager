# 📝 Todo / Task Manager App

A simple and responsive Todo application built using HTML, CSS, and JavaScript.  
It allows users to add, complete, and delete tasks with data saved in browser local storage.

---
Live Demo at:



## 🚀 Features

- ➕ Add new tasks  
- ✔️ Mark tasks as completed  
- ❌ Delete tasks  
- 💾 Data saved using Local Storage  
- 📱 Fully responsive design  
- ⚡ Instant UI updates without reload  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Local Storage API
